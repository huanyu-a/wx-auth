## 📈 文档改进对比

### 更新前
```typescript
// 只有默认模式，无法关闭
WxAuth.init({
  apiBase: 'https://...',
  onVerified: () => { ... }
});
```

### 更新后
```typescript
// 模式1：强制认证
WxAuth.init({
  apiBase: 'https://...',
  required: true,  // 必须认证
  onVerified: () => { ... }
});

// 模式2：可选认证
WxAuth.init({
  apiBase: 'https://...',
  required: false,  // 可选认证
  onVerified: () => { ... },
  onClose: () => {
    // 处理关闭逻辑
  }
});
```
